# 工业生产线故障检测：分层集成学习解决方案

## 项目概述
本项目针对工业大数据场景下的故障检测任务，提出并实现了一种端到端的分层集成学习框架。该方案旨在解决工业数据中普遍存在的类别极度不平衡、高维特征稀疏性及复杂非线性关联等挑战，通过多模型协同与特征增强技术，提升故障检测的精度与鲁棒性。

## 核心架构
采用三级协同架构，实现特征学习、模型建模与结果融合的全流程优化：

1. **特征增强层（Level 0）**  
   基于自编码器（Autoencoder）进行无监督特征学习，仅利用良品样本挖掘高维数据的低维流形结构，过滤噪声并提取鲁棒性潜在特征，缓解高维稀疏性影响。

2. **基模型层（Level 1）**  
   结合两类互补模型刻画数据多维度特征：
   - 梯度提升树（GBT）：捕捉特征间非线性关系与高阶交互效应；
   - FTRL-Proximal：通过L1正则化实现高维稀疏场景下的高效特征选择与线性建模。

3. **模型融合层（Level 2）**  
   采用逻辑回归作为元学习器，对基模型输出进行Stacking融合，自适应优化权重分配，实现多模型优势互补。

## 主要实现

- **分类算法**：逻辑回归、梯度提升树(GBT)、FTRL-Proximal
- **决策树**：CART决策树（支持分类与回归）
- **自编码器**：简单自编码器（用于特征学习）
- **工具类**：交叉验证、模型评估、结果可视化等

## 功能特点

- 全向量化实现，兼顾性能与可读性
- 包含完整的模型训练、评估流程
- 支持特征重要性分析、收敛曲线绘制等可视化功能
- 提供交叉验证与模型融合（Stacking）示例
